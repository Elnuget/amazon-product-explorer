# Project Overview
- **Name**: Amazon Product Explorer
- **Purpose**: React Native Expo app to search and explore Amazon products using RapidAPI endpoints.
- **Tech Stack**: React Native 0.81, Expo 54, TypeScript 5.9, Async Storage (state persistence), React 19.
- **Architecture**: Clean Architecture emphasis with distinct folders for components, screens, services, hooks, types, utils, constants, and navigation.
- **Key Domains**: Category browsing, product search, product details, offers, reviews via Amazon real-time API.
- **Environment**: Node.js 22.19, npm 10.9, Expo CLI 54 required.
