# Suggested Commands
- `npm start` – launch Expo development server.
- `npm run android` – start Expo bundler and open Android emulator/device.
- `npm run ios` – start Expo for iOS (requires macOS).
- `npm run web` – run app in web browser via Expo.
- `npm install` – reinstall dependencies if needed.
- PowerShell basics: `cd <path>`, `ls` to list files, `git status` for repository state.
